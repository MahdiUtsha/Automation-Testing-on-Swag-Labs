# Automation-Testing-on-Swag-Labs
An automation testing is done on Swag Labs website to check its some functionalities (e.g. Homepage Details, Login Functionality, Filter Product Functionality, Add to Cart Functionality, Checkout Functionality &amp; Logout Functionality).

## Features of Swag Labs
**Here are the key features of E-commerce website "Swag Labs" that will be tested:**
- Homepage Functionality
- Login and Registration
- Filter and Sorting Options
- Add to Cart Item
- Booking Workflow
- Checkout Page
- Logout Option

## Technologies Used

- **IDE:**
  - Apache NetBeans

- **Browsers and WebDriver:**
  - ChromeDriver
 
- **Automation Framework:**
  - Selenium WebDriver
 
- **Programming Language:**
  - Java

Make sure you have installed and set up Apache NetBeans, ChromeDriver and Selenium WebDriver on your machine.

- [Install Apache NetBeans](https://netbeans.apache.org/front/main/download/)
- ChromeDriver and Selenium Webdriver files are also given in this repo.
